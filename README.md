# Snake AI using Reinforcement Learning 🐍

## 🚀 Overview
This project implements a Snake AI using Deep Q-Learning (DQN).
The agent learns to play the game and achieves consistent scores of 50–70+.

## 🎯 Key Features
- Ray-cast state representation (32 features)
- Target Network (stable learning)
- Adam optimizer
- Reward shaping

## 📊 Results
- Trained for 12,000+ episodes
- Achieves consistent 55+ scores
